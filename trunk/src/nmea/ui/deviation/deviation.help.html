Deviation Curve

The generic equation of a deviation curve is 

 a + (b.sin(Z)) + (c.cos(Z)) + (d.sin(2.Z)) + (e.cos(2.Z))
where Z is the magnetic heading.

Use the icon to open an already defined deviation curve.

To define your own deviation curve

Use the icon to load logged NMEA data.
The logged data points appear in yellow on the display. Note: The logged values for the heading are Integers.
The points in red are representing the average value of the deviation for each heading. The red points are the points stored for the representation and definition of the deviation curve.

You can:

  • Delete unwanted points. Check the "Del." check box, and use the mouse to drag a rectangle around the points you want to drop
  • Add more points. Check the "Spary" checkbox, and use the mouse (left-click) to spray points on the screen
  • Suggest a smoothed curve ("Suggest" checkbox)
When you hit the "Move points" button, you actually move the red points (defining the deviation curve) on the suggested curve.
